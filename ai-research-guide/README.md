# AI Research Guide

一个全面的 AI 研究指南，涵盖从想法发现到论文发表的完整研究流程。

## 项目结构

- `ai-research-guide/` - 主要指南内容
  - `references/` - 各阶段详细参考文档
    - 01-idea-discovery.md - 想法发现
    - 02-paper-reading.md - 论文阅读
    - 03-idea-validation.md - 想法验证
    - 04-experiment-optimization.md - 实验优化
    - 05-paper-writing.md - 论文写作
    - 06-rebuttal-strategies.md - 反驳策略
    - 07-research-phases.md - 研究阶段
  - SKILL.md - 技能说明

## 使用方法

浏览 `ai-research-guide/references/` 目录下的各个文档，按照研究阶段学习相应的方法和技巧。
